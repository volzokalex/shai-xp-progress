# Shai — XP Progress Prototype

Interactive demo of an XP / Level / Achievement Card system for a learning app.

## Run

Open `index.html` in a browser. No build, no install — React loaded via CDN.

Deployed at: _(URL will be added after publishing)_

## What it demonstrates

- **Plan Configuration** — 12 mission slots, tap to cycle type (Foundation → Builder → Expert). XP values update dynamically so 12/12 always totals 5,950 XP.
- **XP Progress Bar** — level name, within-level XP counter, smooth fill, +XP popup on completion.
- **Mission Map** — 12 zigzag nodes with pulse indicator on next mission.
- **Achievement Cards** — vault of earned cards (Seed / Flame / Crystal by level), badge for unseen cards.
- **Level-Up Overlay** — celebration with particles and animated card reveal.

## Tech

Single-file HTML + React 18 via CDN + Babel standalone.
